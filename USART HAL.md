******
*Пример как отправлять данные через USART*


#### Передача данных:
	После того как сконфигурировал USART нужно создать переменную - буффер со строкой для вывода. После этого с помощью команды отправляем данные в шину.
```c
uint8_t dataOut[] = "Hello data output";

int main(void)
{
	while(1)
	{
		HAL_UART_Transmit(&huart1, dataOut, sizeof(dataOut), 0xFFFF)
	}
	return 0;
}
```

Где: 
- `huart1` - это хэндлер интерфейса **USART1**, цифра может быть и другая, в зависимости от того какой из USART вы выбрали.
- `0xFFFF` - в параметрах функции это Timeout, поставил `0xFFFF` как в гайде, точно не знаю еще как его задавать.
